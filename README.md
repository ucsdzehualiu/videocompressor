# 极速视频压缩 (Video Compressor)

一个基于 FFmpeg Kit 开发的高性能 Android 视频压缩工具app。android studio打开后编译、安装，支持大视频处理、硬件加速以及智能质量检测。

## 🌟 核心功能

- **极速压缩**：优先调用 Android 原生硬件加速 (`h264_mediacodec`)，速度比传统软件压缩快 3-5 倍。
- **智能跳过**：自动检测原视频码率与分辨率。如果视频已经足够精简，则直接极速导出，无需重复重编码。
- **双引擎保护**：当硬件加速不兼容特定格式时，自动无缝切换至 `libx264` 极速软编模式，确保 100% 成功。
- **智能路径选择**：
  - 支持自定义输出文件夹。
  - 默认保存至原视频所在目录，或系统 `Movies/视频压缩` 文件夹。
- **批量处理**：支持一次性选择多个视频进行后台排队处理。
- **大文件支持**：针对 2GB+ 大视频进行了 I/O 优化，防止处理过程中崩溃或产生 0 字节文件。
- **中断支持**：随时可以安全停止当前的压缩任务。

## 🛠️ 技术栈

- **FFmpeg Kit (Full GPL)**：集成最全的音视频处理能力。
- **MediaStore API**：适配 Android 11+ 存储沙盒，支持将视频直接存入系统相册。
- **Material 3**：现代化的 UI 设计语言。

## 🚀 如何开始

1. **选择视频**：点击“选择视频文件”按钮，支持多选。
2. **设定目标**：从下拉菜单选择目标分辨率（1080p, 720p, 480p 等）。
3. **输出设置**：
   - 点击“选择输出保存目录”来指定位置。
   - 若不选，则默认保存至原目录或相册。
4. **开始任务**：点击“开始批量压缩”，进度条将实时显示任务百分比。
